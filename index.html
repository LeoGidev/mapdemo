Este es un Demo del sistema de monitoreo por Ips, ubicadas en en un mapa. Por favor agrege el nombre y las coordenas y los iconos parecerán en el mapa. Le dejamos a continuación una lista de puntos para que pueda probar la web:

  • Plaza Chile: -32.88905258304352, -68.84461802328835
  • Parque General San MArtin: -32.889620280633494, -68.87011899025954
El sistema fué diseñado para monitoreo en empresas "Wireless ISP" (servico de internet por sistema Wireless). Es necesario aclarar que el sistema original es de uso privado y el código esta disponible para quien desee adquirirlo con pago de por medio. El sistema real cuenta con los siguientes item que no son posibles en el demo por estar alojados en un servidor externo y por ser público el código:
  • Sistema en python anailizador de estado por ping
  • Sistema de alerta ante un estado de Down por mensaje de Telegram
  • Conexión entre el php servidor a la base de datos
  • Sistema de login con niveles de usuarios
  • Sistema de "parent" para determinar importancia de equipo caído
